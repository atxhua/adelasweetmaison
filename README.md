# Adela's Sweet Maison — Static Site

A responsive, single‑page site built with semantic HTML, CSS and light JS.

## Deploy on GitHub Pages
1. Create a new repository named `adelas-sweet-maison`.
2. Upload these files (or push via git).
3. In **Settings → Pages**, choose **Deploy from branch**, select the `main` branch and the `/root` folder.
4. Your site will be available at: `https://<your-username>.github.io/adelas-sweet-maison/`.

### Customize
- Replace images inside `assets/images/`.
- Edit contact info and URLs in `index.html`.
- Update `canonical`, Open Graph, and `sitemap.xml` with your GitHub username.
